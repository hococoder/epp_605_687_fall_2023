Welcome to Week 12! This week we'll learn about some important concepts when developing and getting ready to deliver your app

  • Introduction to Accessibility - learn about the built-in accessibility features iOS has that makes your app available to a wider audience
  • Introduction to Internationalization and Localization - iOS is world wide! Learn how to get your app ready for non-U.S. markets
  • Introduction to Instruments - learn how to profile your app in many different metrics by using Instruments
  • Introduction to User Testing - User tests are important when developing code, and here you'll learn about the built in unit testing features of Xcode
  • Programming Assignment 12 - get your app ready for the world by making it accessible and ready for external markets

There is NOT a Week12_Starter or Week12_Final project for this week; example code can be found in the EPExampleApp project in the resource bundle. There is, as usual, a starter project for the homework
