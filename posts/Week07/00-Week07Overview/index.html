Welcome to Week 7! This week we'll dive into Animations, Gestures and Notifications.

  • Intro to Animations - Learn how to animate your views around the screen
  • Introduction to Gestures - Learn about the different type of built in gestures in iOS
  • Introduction to Notifications - Learn about how to implement local notifications and the basics of remote notifications
  • Programming Assignment 7 - Get some animations, gestures and notifications in place for next week's assignment!

There is NOT a Week7_Starter or Week7_Final project for this week; example code can be found in the EPExampleApp project in the resource bundle. There is, as usual, a starter project for the homework
