Updating the Gradebook App

So how can we update our model to include Combine? A simple example would be to add a class to monitor the state of whether we have successfully logged into the app. We'll use this when creating our UI to decide whether we should show the Login screen or the Main landing page of the app.

class UserAuth: ObservableObject {
  
  @Published var isLoggedIn = false
  
  func login() {
    self.isLoggedIn = true
  }
  
  func logout() {
    self.isLoggedIn = false
  }
}

Here, UserAuth is an ObservableObject, and because of that has a property - isLoggedIn - that is decorated with a @Published property wrapper. This means that just before this value changes, it will publish that change to any subscribers. We'll see how to define that subscriber when we talk about SwiftUI State later on in this week's content.

Combine Summary

Combine is Apple's foray into reactive programming. You can setup publishers to transmit values to interested subscribers, and if need be use operators along the way to get the data in just the right form you need, letting you keep your app as up to date as possible.
