Getting Ready

In your last programming assignment, you will add some capabilities to make the app ready for the wider world at large. Start by downloading the HW 12 Starter project in this week's materials. An announcement will go out when the starter project is available.

Downlink HW 12 Starter (link will be active start of week 12)

Assignment Requirements

For this week's homework:

  • Make the Student and Teacher pages accessible - this means making sure the elements on screen have appropriate labels for VoiceOver. Be sure to use the Accessibility Inspector to make sure Voiceover reads them correctly.
  • Localize the app for a language of your choice that iOS supports. This should include any static elements (i.e. the description for an assignment would not be localized since that changes from assignment to assignment; however, the word "Description" or "Max grade" could be localized)

Assignment Submission

  • Zip up the Xcode project folder. This should be the folder that has the .xcodeproj file inside it.
  • Rename the zip to LastName_Homework#.zip
  • Upload the zip to the Programming Assignment 12 area on Blackboard BEFORE 11:59 PM ET on the day it is due (Sunday of Week 12).
  • Be sure to test your app before submitting, so you know it works as intended!

Assignment Rubrics

In addition to the normal grading criteria stated in the syllabus, you will be graded on:

  • Making sure all elements on the Student and Teacher pages have appropriately descriptive accessibility labels
  • Making translations for all requested text
