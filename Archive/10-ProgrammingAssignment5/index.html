Downlink Week 5 Materials (link will be active start of week 5)

Assignment Requirements

For this assignment, you'll be adding some MapKit functionality into the GradebookApp in a few different places. (See the above video for an example)

  • In the StudentView
    • Tapping on the address field should cause a new view to slide in showing that location on a zoomed in map.
    • A pin should be placed at the given location
  • On the CourseClassView:
    • Create a NavigationBarButton that, when tapped, will show a map containing a pin for all students in the class.
    • A pin should be placed at the given location, and when that pin is tapped, the annotation should display the student's name, along with a disclosure button

Note that the locations in the Student objects are given as "City, State", so you will have reverse geocode them to get coordinates for those locations.

Assignment Submission

  • Zip up the Xcode project folder. This should be the folder that has the .xcodeproj file inside it.
  • Rename the zip to LastName_Homework#.zip
  • Upload the zip to the Programming Assignment 3 area on Blackboard BEFORE 11:59 PM ET on the day it is due (Sunday of Week 5).
  • Be sure to test your app before submitting, so you know it works as intended!

Assignment Rubrics

In addition to the normal grading criteria stated in the syllabus, you will be graded on:

  • The Student view show correctly populated map views when tapping on the address fields
  • The CourseClass view has a button that when tapped, shows a map with the locations of all the students in the class, and when the pins are tapped, the user's name should be shown.
