Getting Ready

In your ninth programming assignment, you will add some HealthKit data to the "About" page to allow the user to keep an eye on some important Health data while they complete their assignments. Please download the starter project from the following link to get started:

Homework 9 Starter (link will be active start of week 9)

Assignment Requirements

For this week's homework, please do the following:

  • From the About View, allow the user to view the following from HealthKit:
    • Number of steps for today
    • Number of hours slept in the last 18 hours (this should use the "Asleep" type in the "Sleep Analysis" type, not the "In Bed" type)
    • Amount of water consumed today.
  • Also in the About view:
    • Allow the user to add a water consumption value, in ounces (oz).
    • Notify the user if their sleep was less than 8 hours, their steps are less than 10,000, and their water consumed is less than 64 oz.

Don't forget that on first launch, you'll need to get authorization to access HealthKit (although right at first launch is probably not the best location; but it needs to happen before they access the About page).

NOTE 1: You'll need to add some dummy sample data to the Health app in the Simulator, just like we did in the lecture.

NOTE 2: The University Developer Program doesn't support HealthKit, unfortunately. However, the free developer account you get with your AppleID DOES, so for this assignment, please change the "Team" in the project file (under Signing and Capabilities) to be your individual Team, and when I grade them, I'll change things so it works for me.

You will also need to change your Bundle identifier to something like "edu.jhu.ep.steele.josh.EPGradebookS2022.YourJHEDHere" to differentiate it from the one that I typically use in the class.

This will force you to remove the iCloud and Push Notifications capabilities (ugh) so please do that.

If you have questions about this, let me know.

Assignment Submission

  • Zip up the Xcode project folder. This should be the folder that has the .xcodeproj file inside it.
  • Rename the zip to LastName_Homework#.zip
  • Upload the zip to the Programming Assignment 9 area on Blackboard BEFORE 11:59 PM ET on the day it is due (Sunday of Week 9).
  • Be sure to test your app before submitting, so you know it works as intended!

Assignment Rubrics

In addition to the normal grading criteria stated in the syllabus, you will be graded on:

  • If data is added to the Health App for the given fields, the view in the app should update accordingly
